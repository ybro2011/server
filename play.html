Server Connection −

How to connect:
1. Click "Copy Address"
2. In game: Multiplayer > Add Server
3. Paste using Command + V (Mac)

Press ESC anytime to hide the game.

Game Files Missing

If you see a "404" error, decide which version you want to use:


Option A: Performance (Multi-file)

Drop files (classes.js, assets.epk, etc) in /game/ folder. Rename your game's index.html to game_entry.html.


Option B: Simple (Single file)

Rename your Eaglecraft .html to game.html and put it in the root folder.

Check README.txt for step-by-step instructions.
